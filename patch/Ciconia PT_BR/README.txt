Como instalar o patch

1) Arraste a pasta Ciconia PT_BR para a pasta onde seu
jogo está instalado.

2) Execute o jogo pelo ciconia_phase1.exe na pasta Ciconia PT-BR




------------------------------------------
OBS: Caso tenha saves da versão em INGLÊS,
arraste os arquivos da pasta saves do jogo original
para a pasta saves da versão em português (após iniciar
pela primeira vez)
------------------------------------------



How to Install

1) Drag the folder Ciconia PT_BR to the game directory

2) Run the game using ciconia_phase1.exe inside Ciconia PT_BR
directory.

OBS: If you have previous game saves, drag them from
the ENGLISH game directory's save folder to
Ciconia PT_BR saves folder.
