nsaarc - archives resources into arc*.nsa.

nsaout.exe - extracts the resources from arc*.nsa

nsdec - extracts the script from any given nscript.dat
obs: rename "pscript" to "nscript" to extract.

nscmake - takes a 0.txt script and converts to nscript.dat
